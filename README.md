# health-check-
基于临床指南的智能体检推荐系统。纯前端零上传，2分钟生成个性化清单。
本项目是一款基于临床指南的智能体检推荐系统，只需回答 6 个核心问题，即可为您生成精准的个性化体检清单。同时支持查询大部分体检指标异常情况。

项目亮点：

隐私安全：纯前端运行，数据零上传，所有档案仅保存在您的浏览器本地。
算法驱动：内建基于年龄、性别、慢病、家族史的分层推荐引擎。
指标解读：涵盖 40+ 常见检查项目，提供详尽的正常参考范围与异常临床意义。
开箱即用：无需后端，无需注册，打开网页即可使用。


A smart health checkup recommendation system based on clinical guidelines. Answer 6 questions to get a personalized checkup list.

Privacy First: 100% client-side, zero data upload. Your data stays in your browser. 
Algorithm Driven: Rule-based engine using age, gender, chronic diseases & family history.
Deep Interpretation: Detailed reference ranges and clinical significance for 40+ common tests.
No Backend Needed: Just open and use.
